# Invent

## Action

1.  When drawn, gain Lowest Priority.
1.  THEN: Create, Modify OR Remove a **Concept** (DC 13 - 2 per relevant <InlineDefine capitalize term='active'/> Detail)

## Keywords

-   **Create** - Create a new concept. DC 13 -2 for each relevant <InlineDefine capitalize term='active'/> Detail.
-   **Modify** - Change the rules of an existing concept. DC 13 -2 for each relevant <InlineDefine capitalize term='active'/> Detail.
-   **Remove** - Delete an existing concept. DC 13 -2 for each relevant <InlineDefine capitalize term='active'/> Detail.
